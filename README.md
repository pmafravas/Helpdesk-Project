# Helpdesk Project

Um projeto pessoal realizado em .NET 10 com interface web Blazor Web App de um sistema de Helpdesk.

O sistema de gestão de tickets de suporte técnico é focado em alta disponibilidade e processamento assíncrono. O diferencial do projeto é demonstrar domínio sobre arquiteturas baseadas em eventos, retirando a carga de processamento pesado da thread principal do usuário e fornecendo feedback em tempo real.

## Stack Tecnológica
- **Backend e Frontend**: .NET 10 (C#), Blazor (Server ou Web App) e SignalR para WebSockets.
- **Banco de Dados**: MariaDB (Relacional) com Entity Framework Core (ORM).
- **Mensageria**: RabbitMQ (Message Broker).
- **Identidade**: ASP.NET Core Identity.
- **Hospedagem Local**: Docker Compose.
